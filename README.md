## 프로젝트 소개

- 해당 프로젝트는 한남대 컴퓨터공학과 동아리원들을 위해 제작한 웹 서비스 입니다.
- 공지사항, TIL, 랭킹, SHOP, Event, MyPage, AdminPage로 이루어져 있습니다.
- github 활동을 장려하기 위해 github만으로 로그인이 가능하게 구현하였습니다.
